# 校园课程表智能提醒工具
基于Streamlit开发的课程表提醒工具，支持Excel导入、调课管理、实时提醒，部署于Streamlit Community Cloud。

## 使用说明
1. 访问部署链接：[你的链接（部署后补充）]
2. 侧边栏设置开学日期，上传Excel课程表（模板见下方）；
3. 调整提前提醒时间，可添加调课信息；
4. 主界面查看今日/本周课程，右侧实时提醒即将上课的课程。
